# Documentation du Projet Portfolio

Ce dossier contient toute la documentation relative au développement du portfolio personnel.

## Structure

- `journal/` : Suivi quotidien des activités
- `references/` : Documentation technique et références
- `ressources/` : Fichiers et ressources utiles
- `journal-de-bord.md` : Vue d'ensemble de l'avancement du projet

## Comment utiliser

1. Consultez le `journal-de-bord.md` pour un aperçu global
2. Utilisez les dossiers thématiques pour des informations détaillées
3. Mettez à jour régulièrement la documentation au fur et à mesure de l'avancement
